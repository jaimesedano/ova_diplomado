Hardware y Software

Hardware: Conocido como las partes físicas del computador ejemplo: teclado, mouse, monitor. (Tomado de: http://www.aliat.org.mx/BibliotecasDigitales/sistemas/Arquitectura_computadoras_I.pdf )

Software: conjunto de instrucciones que dirigen al hardware. Asimismo, es un conjunto de instrucciones que realizan una tarea específica denominada programa. (Tomado de: http://www.aliat.org.mx/BibliotecasDigitales/sistemas/Arquitectura_computadoras_I.pdf )
